- Compatible Bitcoin Software Wallets -

BTChip and HW-1 main goals are to provide hardware wallets and not the full software client experience, user interface and options... There are already many great software clients we just want to be compatible with and we'll do our best to help every software wallet developper to make their software compatible with HW-1. Here are the wallet software providers we already support:

  • GreenAddress: available for Chrome, Android Google Play and F-Droid
  • ELECTRUM: available in development version.
  • KRYPTOKIT: unofficial release (deprecated)
  • More to come soon... If you got your own software wallet, feel free to contact us
